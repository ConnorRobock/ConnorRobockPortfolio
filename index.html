Project 2 - "Find Me A Meal App"

This paired project was completed by Akalpit Sharma and myslef. The Find Me A Meal App is designed to allow the user to search recipes that include a particular ingredient or select the random button to provide the user with a random recipe.

This project showcased our ability to use APIs while integrating JavaScript with html and css. This was a very fun project that allowed us to use some creativity to accomplish our goals. It was great to work in pairs and learn what it is like successfully code as a team.
